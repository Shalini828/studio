# 🚀 StudyAgent

> Your phone thinks. Your laptop executes.

StudyAgent is an AI-powered multi-agent study companion built for the iQOO Hackathon. It helps students generate personalized study plans, discover learning resources, organize notes, track focus, and improve productivity using AI orchestration.

---

## ✨ Features

### 🔍 Research Agent
- Finds the best learning resources
- Suggests courses, videos, articles, and practice platforms
- Curates topic-wise references

### 📝 Notes Agent
- Extracts important concepts
- Generates concise study notes
- Highlights key revision points

### 📅 Schedule Agent
- Creates personalized study roadmaps
- Generates day-wise learning plans
- Balances practice and revision sessions

### 📄 Summary Agent
- Produces executive summaries
- Creates quick revision sheets
- Generates interview-focused takeaways

### 🛡️ Focus Guardian
- Calculates productivity score
- Monitors study consistency
- Suggests focus improvements

## 🎯 Problem Statement

Students often struggle with:
- Finding quality resources
- Organizing study material
- Creating effective schedules
- Maintaining focus

StudyAgent solves these challenges through a multi-agent AI workflow that transforms a simple study goal into a complete learning plan.

---

## 📸 Demo Workflow

1. Enter a study goal
2. AI Research Agent gathers resources
3. Notes Agent extracts key concepts
4. Schedule Agent creates a roadmap
5. Summary Agent generates revision notes
6. Focus Guardian evaluates productivity
