# **App Name**: StudyAgent

## Core Features:

- Intelligent Orchestration Engine: An AI tool that coordinates the workflow between the Planner, Research, and Note agents based on a user's study goal.
- Live Workflow Visualization: Animated progress UI showcasing agents as they sequentially process data and move from 'idle' to 'complete'.
- Research & Resource Synthesis: AI agent that utilizes Gemini to discover and list curated web resources and reference materials.
- 7-Day Curated Roadmap: Generation of a structured study schedule and specific note topics tailored to the subject depth.
- Focus Guardian Module: AI-powered productivity agent that provides behavioral focus suggestions based on the complexity of the task.
- Neon Studio Workspace: A glassmorphic dashboard built with Shadcn UI components for high-legibility resource display and revision management.
- Gemini Integration Layer: Edge-compatible Next.js API routes that handle low-latency processing for agent responses.

## Style Guidelines:

- A dark futuristic scheme with a deep space-black background (#0B080F) based on desaturated indigo.
- Primary: Electric Neon Purple (#A855F7) to symbolize AI activity; Accent: Bright Cyber Indigo (#6366F1) for gradients and UI indicators.
- Font pairing: 'Space Grotesk' for technical headers and branding to provide a scientific feel; 'Inter' for readable content blocks.
- High-end Glassmorphism featuring semi-transparent cards with 2px blurred borders and 24px border radii.
- Orchestrated Framer Motion transitions with staggered entry animations for agent result cards.
- Minimalistic line icons with subtle purple glows and 'pulse' animations when an agent is active.
